本工具用于计算视频的时间信息（TI，也称为时间复杂度），以及空间
信息（SI，也称为空间复杂度），以上两个指标取自于ITU-R BT.1788标准。

This software is used to calculate raw video's TI (Temporal perceptual 
Information) and SI (Spatial perceptual Information) defined in 
ITU-R BT.1788.
