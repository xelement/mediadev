本程序实现了对内存中的视频数据的读写。包含两个工程：
simplest_ffmpeg_mem_player：播放内存中视频数据的播放器。
simplest_ffmpeg_mem_transcoder：转码内存中数据的转码器。


This software simplest example to use FFmpeg to read (or write) 
from memory.

It contains 2 project:
simplest_ffmpeg_mem_player：Video Player that play video 
data in memory.
simplest_ffmpeg_mem_transcoder：Video Converter that convert 
video data in memory.
